# Cricket Performance Analytics Dashboard

# Project Overview 🏏
This project presents an interactive Cricket Performance Analytics Dashboard built using Power BI. It provides deep insights into player performance, team statistics, and match outcomes to support data-driven decision-making in cricket analysis.

**The dashboard is designed to help analysts, coaches, and cricket enthusiasts evaluate performance trends, compare players, and identify strengths and weaknesses across matches and tournaments.**

# Objectives 🎯
- Analyze player performance (batting & bowling)
- Track match outcomes and trends
- Identify top-performing players
- Compare teams based on key metrics
- Enable interactive filtering for better insights

---

# Tools & Technologies 🛠
- **Power BI** – Data visualization & dashboard creation
- **Excel / CSV Dataset** – Data source
- **DAX (Data Analysis Expressions)** – Calculations & KPIs
- **Data Modeling** – Relationships between tables

---

# Key Features 📌
1. **Interactive Dashboard**
- Dynamic filters (player, team, match type, year)
- Drill-down capabilities for detailed analysis
2. **Batting Performance Analysis**
- Total Runs
- Strike Rate
- Batting Average
- Boundaries (4s & 6s)
3. **Bowling Performance Analysis**
- Wickets taken
- Economy rate
- Bowling average
- Overs bowled
4. **Player Comparison**
- Compare multiple players side-by-side
- Identify consistent performers
5. **Match Insights**
- Win/Loss analysis
- Team performance trends
- Venue-based performance
6. **KPI Cards**
- Total Matches
- Total Runs
- Total Wickets
- Best Performer

---

# Dashboard Components 📊
- **Bar Charts** – Player comparisons
- **Line Charts** – Performance trends over time
- **Pie/Donut Charts** – Contribution analysis
- **KPI Cards** – Quick summary metrics
- **Slicers** – Interactive filtering

---

# How to Use 📂
1. Download the .pbix file from this repository
2. Open in Power BI Desktop
3. Explore dashboards using filters and slicers
